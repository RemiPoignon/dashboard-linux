Dashboard-linux

Dashboard-linux est un tableau de bord de vérification des ressources utilisées par votre serveur linux en temps réel. Très simple à installer, il est développé en php et bash, il vous suffit de copier le répertoire dans votre répertoire www d'apache (ou de lui créer un VHost dédié).

Fonctionnalité

  • Jauge CPU utilisé
  • Jauge RAM utile utilisé
  • Jauge Espace disque utilisé
  • Informations générale sur le serveur (nom, ip, uptime)
  • Information sur le CPU (nombre de cœurs)
  • Information sur la RAM (quantité max, utilisé, en cache, libre + SWAP)
  • Information sur l'espace disque (max, utilisé, restant)
  • Liste des utilisateurs connecté en ssh (utilisateur, date de connexion, ip)

Installation

  1. Télécharger le package
  2. Créez un virtual-host Apache pointant vers le dossier du package (ou copier le dans votre www).

Support

  • PHP 5 (avec possibilité d'utiliser la commande shell_exec)
  • Apache 2

Test

Testé avec Ubuntu Server 13.4
